# nest-n

Proyek sederhana dan ramah untuk memulai pengembangan dengan Nest (atau kerangka kerja serupa). Tujuannya: mudah dibaca, mudah diubah, dan menyenangkan untuk dikembangkan.

## Fitur
- Struktur modul yang jelas
- Konfigurasi dasar untuk development
- Contoh endpoint dan pengujian ringan

## Instalasi
1. Pastikan Node.js dan npm/yarn terpasang.
2. Instal dependensi:
```bash
npm install
# atau
yarn
```

## Menjalankan
- Development:
```bash
npm run start:dev
# atau
yarn start:dev
```
- Build:
```bash
npm run build
```
- Test:
```bash
npm run test
```

## Struktur proyek (singkat)
- src/ — kode sumber
- src/modules — modul aplikasi
- src/common — utilitas dan middleware
- test/ — pengujian

## Kontribusi
- Buka issue untuk diskusi fitur/bug.
- Kirim PR ringkas dengan deskripsi perubahan.
- Sertakan test bila perlu.
